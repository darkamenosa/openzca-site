openzca

Free, open-source Zalo CLI wrapper built on zca-js. Use openzca ... (alias zca ...) for auth, messaging, groups, friends, and profile automation.

GitHub npm package

Install (npm)

npm install -g openzca
openzca --help

Install Script (macOS/Linux)

curl -fsSL https://openzca.com/install.sh | bash

Install Script (Windows PowerShell)

irm https://openzca.com/install.ps1 | iex

Quick start

openzca auth login
openzca auth status
openzca msg send <userId> "hello"

Primary domain: openzca.com. Secondary domain can point/redirect to the same site.
